Speed Site

This site was created at speed with a few rules. The main rule is to use grids for layout. The layout is kind-off based on the default Joomla theme.

For spacing, prioritise using line-height over margin where possible. Prioritise using padding over margin.

Insights

Since I used HTML very based of the default Joomla theme, there were a lot more container div's than necessary. This left me with most of the grids with only two items, rendering the advantage of grids in this example pointless - oh well, it was good practice.

What I did find quite interesting was using similar SVG's as fixed backgrounds, with only their colour differing. It created a pretty cool parallaxy effect.
